# GenAI Prompt Engineering using LangChain

## Overview
This project implements a mini prompt engine using LangChain PromptTemplates.

## Features
- Dynamic Prompt Templates
- Multi-input prompts
- Prompt variations (Teaching, Interview, Storytelling)
- ChatPromptTemplate system
- Input validation layer
- Reusable prompt system

## Tech Stack
- Python
- LangChain
- Jupyter Notebook
